
# Introduction
Bienvenue dans le guide d'utilisation de SmartVacc, l'outil conçu pour simplifier et optimiser la campagne de vaccination vétérinaire. Cet outil vous aide à enregistrer efficacement les données des animaux vaccinés, générer des rapports, générer des statistiques et bien plus encore, le tout en réduisant le risque d'erreurs.
